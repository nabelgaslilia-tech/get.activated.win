# running the script

go to https://github.com/nabelgaslilia-tech/get.activated.win/releases to get the latest .zip or .7z file then extract.

go to windows powershell or terminal (if you have windows 11 installed in your laptop, computer, or PC.)

navigate to where you downloaded the .zip or .7z file.

type this into the prompt: `powershell -ExecutionPolicy Bypass -File .\myscript.ps1`.

or just run the .bat file if you're a lazy sloth

## editing the script

if you wanna see or edit the source code, right-click the .ps1 file, tap "Open With", and select "Notepad".

you can copy the entire source code by pressing 2 shortcuts: Ctrl + A + Ctrl + C
